# Tic Tac Toe Game

![page-header](imgs/githubImg.png)

## Description 
This is simple web game of Tic Tac Toe made with Javascript. 

## Features
- Score counter and reset 
- Can be played both on PC or Mobile layout
- Win state modal

## Try it out: 
https://tykejmi-tictactoe.netlify.app

## Technologies used:
- HTML5
- CSS3
- JS
